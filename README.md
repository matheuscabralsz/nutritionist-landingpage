# karennutri Landing

Production landing page for an online nutrition practice. Built and deployed solo.

**Stack:** Next.js 14 (App Router) · React 18 · TypeScript · Tailwind CSS · lucide-react

**Highlights**
- Conversion-focused single-page site with custom UI components (no template)
- Fully typed, server-rendered, mobile-first
- Clean component structure under `components/landing` and `components/ui`

**Full Demo**
https://karennutri.vercel.app

```bash
npm install
npm run dev
```
